# 高德地图API集成说明

## 概述

本项目已集成高德地图API，提供3D立体地图效果，适用于视障智能导航系统。

## 功能特点

- 3D立体地图视图
- 建筑物显示与动画效果
- 实时位置标记
- 2D/3D视图切换
- 地图控件（缩放、旋转、工具栏等）
- 自动定位功能
- 响应式设计

## 配置步骤

### 1. 获取高德地图API密钥

1. 访问 [高德开放平台](https://console.amap.com/dev/key/app)
2. 注册/登录账号
3. 创建新应用
4. 获取API密钥

### 2. 获取安全密钥

1. 在高德开放平台控制台中
2. 选择您的应用
3. 在"获取Key"页面中找到"安全密钥"
4. 复制该密钥用于配置

### 3. 配置API密钥

编辑 `src/config/amap.js` 文件，将 `AMAP_CONFIG.key` 的值替换为您自己的API密钥，并添加安全密钥：

```javascript
export const AMAP_CONFIG = {
  // 高德地图API密钥（请替换为您自己的密钥）
  key: '您的高德地图API密钥', // 替换这里
  // 高德地图安全密钥（请替换为您自己的安全密钥）
  securityCode: '您的安全密钥', // 替换这里
  // ...其他配置
}
```

### 3. 配置Web服务

在高德开放平台的应用设置中，添加Web服务，并设置以下信息：

- 服务名称：Web服务
- 服务类型：Web服务
- 白名单：`*` 或您的域名（开发阶段可使用 `*`）

## 使用说明

### 地图控件

- **缩放控件**：右下角，用于缩放地图
- **工具栏**：右下角，包含平移、缩放等工具
- **3D控制条**：右上角，用于调整3D视角
- **定位按钮**：左下角，用于获取当前位置

### 功能按钮

- **重置视角**：将地图恢复到默认视角和位置
- **2D/3D视图切换**：在2D和3D视图之间切换

### 自定义配置

您可以在 `src/config/amap.js` 文件中修改以下配置：

- `defaultCenter`：默认地图中心点
- `defaultZoom`：默认缩放级别
- `viewMode`：默认视图模式（'2D'或'3D'）
- `pitch`：3D视图下的俯视角度（0-83）
- `rotation`：地图旋转角度
- `mapStyle`：地图样式
- `skyColor`：天空颜色
- `showBuildingBlock`：是否显示建筑物
- `buildingAnimation`：是否启用建筑物动画

## 故障排除

### 地图无法加载

1. 检查API密钥是否正确配置
2. 确认网络连接正常
3. 检查浏览器控制台是否有错误信息

### 定位功能不工作

1. 确保浏览器支持定位功能
2. 检查是否已授予定位权限
3. 尝试使用HTTPS访问（某些浏览器要求）

### 3D效果不明显

1. 调整 `pitch` 值（建议40-60）
2. 确保在支持WebGL的浏览器中访问
3. 尝试缩放地图到城市级别
